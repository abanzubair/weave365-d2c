# 🧵 SAARANG — Saree E-Commerce Design System

## 🌌 Brand Vision
A cinematic, modern saree shopping experience blending tradition with Gen Z minimalism.  
Not just clothing — **a visual narrative of fabric, light, and motion.**

---

## 🎨 Design Language

### Core Vibe
- Cinematic
- Warm luxury
- Soft shadows + depth
- Editorial fashion aesthetic

### Color Palette

| Role            | Color        | Hex       |
|-----------------|-------------|-----------|
| Primary         | Deep Maroon | #4A1F1F   |
| Accent          | Gold        | #C8A96A   |
| Background      | Cream       | #F5EFE7   |
| Dark Base       | Charcoal    | #1A1A1A   |
| Text Primary    | Black       | #111111   |
| Text Secondary  | Soft Gray   | #6B6B6B   |

---

## 🔤 Typography

### Headings
- Serif (Elegant, editorial)
- Example: Playfair Display / Cormorant

### Body
- Sans-serif (clean, modern)
- Example: Inter / Poppins

### Style Rules
- Large hero text (48px–72px desktop)
- Generous letter spacing
- Minimal but impactful
